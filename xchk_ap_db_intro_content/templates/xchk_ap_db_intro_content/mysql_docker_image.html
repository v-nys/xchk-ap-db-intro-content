Schriftelijk materiaal

Je vindt een Docker image voor MySQL terug op deze officiële pagina.

Je hoeft dit nog niet te begrijpen. Het volstaat dat je weet dat je op deze pagina moet zoeken als je ooit een MySQL in een container wil opzetten op een andere manier dan wij het in de les zullen doen.

In de les gebruiken we namelijk het exacte commando: docker run --name mijn-mysql-container -e MYSQL_ROOT_PASSWORD=mijn-mysql-container-wachtwoord -p 3306:3306 -d mysql:8.0.21

Dit zorgt ervoor dat je een MySQL server uitvoert met:

  • versie 8.0.21
  • één gebruiker, namelijk root
  • wachtwoord voor root: mijn-mysql-container-wachtwoord
  • toegankelijk voor clients via poort 3306

Als je deze container ooit stopt en hem later wil herstarten, dan kan je gewoon schrijven: docker start mijn-mysql-container
