Schriftelijk materiaal

MySQL is een veelgebruikt relationeel DBMS. Het is open source, wat betekent dat iedereen de broncode kan bekijken. Het heeft veel features, uitstekende documentatie en je vindt makkelijk ondersteuning op het web.

Als je met MySQL werkt, werk je eigenlijk niet met één programma, maar met twee programma's: een client en een server. Deze twee programma's kunnen op twee verschillende machines draaien, maar ze kunnen ook op dezelfde machine draaien.

De client is het programma dat je gebruikt om het DBMS aan te spreken, maar is geen onderdeel van het DBMS zelf. Voor MySQL is dit typisch het programma MySQL Workbench of de MySQL Command Line Interface. Het kan ook een programma zijn dat je zelf schrijft. Via de client verstuur je instructies, maar de client voert zelf de instructies niet uit.

De server bevat het DBMS zelf. Het DBMS voert de instructies uit die het ontvangt. Omdat de client en de server op twee verschillende locaties kunnen zitten, is het nodig de client met de server te verbinden door bij het opstarten van de client het adres van de server mee te geven.

Vergelijk het met websites. Met je browser verbind je met een bepaald adres en haal je de nodige gegevens af om iets te zien op je scherm. Miljoenen bezoekers kunnen tegelijkertijd op dezelfde website zitten en als je je browser sluit, is de website weg op jouw machine. Jij en de andere bezoekers zijn dan allemaal clients. De machine die je de gegevens stuurt die je nodig hebt om de website op je scherm te krijgen, d.w.z. de machine die het eigenlijke werk doet, dat is de server.
